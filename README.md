# Walmart-Customer-Retention-Analysis
This project focuses on understanding customer retention, churn behavior, and repeat purchase patterns for Walmart using interactive Power BI dashboards. The analysis identifies key factors that influence customer loyalty and provides actionable insights that can help improve revenue, customer engagement, and long-term growth.

## Project Overview 
Retail businesses like Walmart thrive on repeat customers. Retained customers spend more, return more frequently, and contribute significantly to revenue stability.
This Power BI dashboard analyzes customer behavior using transaction data to reveal:
  - How many customers are returning
  - Who is at risk of churning
  - Which products drive loyalty
  - Month-on-month retention trends
  - Branch-level performance differences

The report helps business teams make data-driven decisions to improve loyalty programs and enhance customer experience.

## Objectives 
- Measure Customer Retention Rate
- Identify Churned Customers
- Perform Cohort Analysis
- Analyze purchase frequency and spending behavior
- Explore retention by product categories and store branches
- Provide insights for actionable business strategies

Key Matrics (KPI's)
- Churn Rate
- Retention Rate
- Repeat Rate
- Customer Lifetime Value (CLV)
- Purchase Frequency


## Analysis Performed 
- Churn & Retention analysis
- Repeat Purchase analysis
- Promotion & Loyalty Impact analysis
- Store & Channel Performance vs Retention
- Customer Lifetime Value (CLV) Analysis

## Key Insights 
- Returning customers contribute higher average revenue than new customers.
- High CLV customers are mostly the loyal customers with the most recent purchases.
- Senior age group impacts the purchase frequency.
- The impact of promotions are mostly seen for new users.
- Apparel and Groceries categories show stronger loyalty.
